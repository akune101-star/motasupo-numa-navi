# Codexに毎週投げる更新依頼文

モタスポ沼ナビの今週分データを更新してください。

対象カテゴリ:
- F1
- WRC
- WEC
- SUPER GT
- スーパーフォーミュラ
- フォーミュラE
- NASCAR

作業内容:
1. 各カテゴリの公式順位表または信頼できる情報源を確認
2. `data/standings.json` の上位5件を更新
3. `data/stories.json` の「今季のストーリー」「今見る理由」「次に見るポイント」を更新
4. `sourceName`、`sourceUrl`、`updatedAt` を必ず更新
5. 写真・公式ロゴ・長文転載は禁止
6. 初心者にも分かる文章にする
7. `npm run validate` を実行
8. 問題なければPull Requestを作成

PR本文には以下を書いてください:
- 更新したカテゴリ
- 参照した主な情報源
- 未確認または注意が必要な点
