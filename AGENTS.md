# AGENTS.md / CODEX.md

このリポジトリは「モタスポ沼ナビ」という初心者向けモータースポーツ観戦ガイドです。

## 目的

F1以外のモータースポーツも、順位表だけでなく「今の物語」「注目人物」「歴史」で理解できるようにする。

## 更新ルール

- 更新時は必ず公式サイト、公式順位表、または信頼できる情報源を確認する。
- `data/standings.json` には `updatedAt` と `sourceUrl` を必ず入れる。
- 順位表示は上位5件まで。
- 写真、公式ロゴ、映像、長文転載は禁止。
- 文章は初心者向けにする。
- 断定できない情報は書かず、「未確認」または空欄にする。
- 公式順位表へのリンクを残す。
- 更新後は `npm run validate` を実行する。
- 変更はできればPull Requestとして出し、本文に「更新したカテゴリ」「参照元」「注意点」を書く。

## 週1更新時の作業

1. F1 / WRC / WEC / SUPER GT / スーパーフォーミュラ / フォーミュラE / NASCAR の公式順位表を確認
2. `data/standings.json` の上位5件を更新
3. `data/stories.json` の今季ストーリー、今見る理由、次に見るポイントを更新
4. `sourceName`、`sourceUrl`、`updatedAt` を更新
5. `npm run validate` を実行
6. PRを作成
